The three architectural commitments
What it means
to be government-grade.
01

On Indian foundation models

STT on AI4Bharat IndicConformer (IIT Madras). LLM interpretation on Sarvam-1. TTS on Indic-Parler-TTS (production swap to Sarvam Bulbul for native Kannada). Self-hosted. No PII to hosted LLMs, ever. Bhashini wired in as the government-blessed fallback path.

02

Audit-logged, end to end

Every state transition — Heard → Interpreted → Verified → Acted → Escalated — writes an immutable event with timestamps, model versions, payload hashes, audio chunk references. Officer-retrievable per call ID in ≤ 5 seconds. WORM storage. 7-year retention configurable per tenant.

03

No source-system modification

Voca is a parallel assistive layer. The 1092 telephony stack stays untouched. We tap call audio via SIP fork or LiveKit's media bridge and send interpretation events to the agent console. Existing systems are unmodified. Procurement is a vendor add, not a re-platform.

For an auditor
Every decision retrievable.

An IAS officer reviewing a specific call should not need our help to find the trail. Voca's audit posture is designed so the entire decision chain — the input audio, the STT confidence, the interpretation, the verification turn, the citizen's response, the action taken — is a single query away.

The audit query is one line.
The result is the full call timeline.

Given any call ID, a government auditor (with appropriate role) sees the complete sequence of state transitions:

$ voca audit show CALL_2026_05_06_AB47CF
  t=0.000s  HEARD      stt:indicconformer-v3.2  conf=0.78
  t=0.612s  INTERPRETED llm:sarvam-1-grievance-ft-2026q2  risk=HIGH conf=0.71
  t=0.640s  BYPASSED    rule:distress>0.65 AND risk=HIGH
  t=0.701s  ACK         tts:bulbul-kn-IN  dialect=Mandya register=informal
  t=0.701s  ESCALATED   agent_id=4 women_cell=true
  t=2.487s  ACTED       agent_pickup=true context_loaded=true

Audio chunks are referenced by hash, fetchable from WORM storage. Model versions are pinned per row, so a re-run of the interpretation against historic audio is reproducible.
